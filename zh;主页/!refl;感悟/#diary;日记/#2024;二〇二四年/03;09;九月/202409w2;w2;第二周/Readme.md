## 明天的每日一题

计划额外把明天的[每日一题](https://leetcode.com/problems/find-the-longest-substring-containing-vowels-in-even-counts/)也做一下子. 我第一眼看到题目, 感觉应该是"slide window", 但是其实不是哈哈, 想错了方向一会子. 我关了灯, 躺在床上想一会儿这个问题. 我想到感觉今天去"County Market"还勾起了我一点子购物的想法. 我一起身, 想再去吃点今天买的哈根达斯樱桃冰激凌, 突然就想到了解决方案. 虽然大部分还是在床上想到的, 但是一起身感觉突然完全明白了还挺有趣的.

其中还发生了个小 bug. 就是我忘记设置空字符应该是也是一个符合要求的字符串哈哈.

```rust
/// 大致就是说一段[String]里, 其中最长的一段,a,e,i,o,u都是偶数个的连续字符的长度是多少
pub fn find_the_longest_substring(s: String) -> i32 {
    #[inline]
    fn get_vowel_char_idx(c: char) -> Option<usize> {
        match c {
            'a' => Some(0),
            'e' => Some(1),
            'i' => Some(2),
            'o' => Some(3),
            'u' => Some(4),
            _ => None,
        }
    }
    let mut all_vowels_count_even_bitflags: u8 = 0;
    let mut ans_len: usize = 0;
    let mut prev_info: [Option<usize>; 32] = [None; 32];
    prev_info[0] = Some(0); // 这个初始比较重要哈哈
    for (i, c) in s.char_indices() {
        if let Some(vowel_i) = get_vowel_char_idx(c) {
            let pos_bitmask: u8 = 1 << vowel_i;
            all_vowels_count_even_bitflags ^= pos_bitmask;
        }
        let prev_cell_mut_ref = &mut prev_info[all_vowels_count_even_bitflags as usize];
        if let Some(prev_i_mut_ref) = prev_cell_mut_ref {
            ans_len = ans_len.max(i + 1 - *prev_i_mut_ref);
        } else {
            *prev_cell_mut_ref = Some(i + 1); // 如果说空是在位置0的话
        }
    }
    ans_len as i32
}
```
