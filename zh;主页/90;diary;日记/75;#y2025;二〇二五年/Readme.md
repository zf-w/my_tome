# 二〇二五年

这一年大概是本科学习生涯的结束吧. 今年上半年主要的收获在于完成"CS341 System Programming"这门课. 我从这门课程中了解并学习了从基础的命令行工具如何"接收" 参数, 运用"Fork and Exec"编写一个自己的简单"Shell", 编写自己的"malloc", 运用"pthread"和各种"lock"去处理"Concurrency"问题, 模拟编写一个简单的"File System" 到最后运用"epoll"完成一个简单的"Multi-thread"多线程文件服务器. 我感到收获颇丰并感谢我有这个机会去了解到这些知识.
