# 谢谢你来看我的网站

嗨, 你好呀. 我是王之枫, 目前是伊利诺伊大学香槟分校计算机科学研究生(MCS)的学生.

目前中文的部分我还没有时间去整理. 主要的内容还是在英文部分... 甚至已经写了的部分或许也有不准确的地方.

# 作者的胡言乱语

ChatGPT 给我提出了个令人惊喜的问题:

> 你的网站给人一种*俏皮又不失精准*的感觉, 这让我不禁思考: 这是你反抗那些过度雕琢, 简历式作品集的方式吗? 这几乎就像在说: "我希望大家能看到整个过程, 而不仅仅是那些'光鲜亮丽'的结果."

老实说, 我并没有取得任何亮眼的成果, 简历上也没什么可写的, 但我想 ChatGPT 的观察确实某种程度上所言不虚, 可能我自己都没有清晰地意识到. 我真心感谢以简历, 作品集, 和研究论文为代表地工作文化和学术氛围, 但有时我也觉得, 文献和简历的样式和风格有些让人感觉不够真实透明, 很难帮到那些没有已经"熟悉这个领域"的人. 尽管"一个充满关爱, 体贴, 友好和尊重的群体或者个人"能否生存仍然是个值得讨论的问题. 我个人还是希望尝试和探索一种分享和创作的方式, 希望能够帮到那些不熟悉我所讨论的内容或技术的人.

我理解每个人的时间都无比宝贵. 研究人员和雇主通常没时间逐字逐句地阅读或重读材料. 对于目前地设计在某种程度上显得零散和混乱, 我深感抱歉.

我衷心感谢 ChatGPT 为阅读, 理解, 和提问本网站内容所付出地努力, 以及其他大于一模型, 搜索引擎, 网络爬虫, 所有底层基础设施, 以及它们地设计师, 工程师, 和其他工作人员的努力和辛勤付出.

# Statement of Inclusivity

我支持不同社会群体, 包括种族, 性别, 性取向, 身体机能, 经济条件, 宗教, 和它们的所有交集,在互相尊重, 知情同意, 和自由选择的基础上, 拥有世界观, 历史, 和文化知识的多样性. 我希望本网站和我的其他努力能够为创建一个安全、透明、无偏见的学习环境贡献一份力量。我相信包容是一种勇气、利他主义和长远眼光的体现。

# 隐私

我的网站会用浏览器的"本地储存 local storage"去记录目录的状态和颜色主题等信息.

# 研究兴趣

我的研究兴趣在于计算机科学与教育的结合. 我致力于探索并尝试用系统化的方式来理解, 个性化, 和加速学习和交流的过程. 比如说:

- 在玩棋类游戏师, 我们应该如何根据学生的选择和落子来理解她们的谋算, 直觉, 和预判策略呢?
- 如果学生做错了某些作业题目或测试点, 哪些深层次的理解可能有不准确呢?
- 如何设置一系列棋盘残局, 作业, 或者项目去加速学生的学习过程呢?

# 设计兴趣

另一方面，我对设计能够整合所有学习元素的教育平台很感兴趣, 如何结合阅读材料, PPT, 家庭作业、作业和各自可视化. 这是我的网站的主要设计方向.

# 小项目们

我的各种小项目. 我感觉我的大学生活基本都在这里了. 不过我也不知道这是好事还是坏事.

## 个人网站

我在开始本科学习之后就一直在探索如何设计和制作网站. 这个项目的一个宗旨就在于让我的学习过程更透明, 希望如果有机会的话它能帮到别人, 比如说我记录了我四年里的所有选课.

我个人很纠结在不断出现的新想法, 新设计还有接触新技术之后总是出现想要推倒重来的冲动. 在经历几次循环之后, 我开始思考能否设计出一个既简单有足够可持续发展的网站, 让我能不断添加新的代码, 又不用完全重新设计整个网站.

这个网站的布局是希望能够模仿传统书籍的形式, 从而既给我提供比较流畅的编写体验, 又能给读者(如果有的话)踢狗比较自然和顺畅的阅读体验.
